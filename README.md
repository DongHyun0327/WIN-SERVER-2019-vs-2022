# Windows Server Group Study

Windows Server 2019 and Windows Server 2022를 주차별로 학습하고 비교하는 그룹 스터디 저장소입니다.

## Structure

```text
windows-server-study/
├─ README.md
├─ .gitignore
├─ .editorconfig
├─ docs/
├─ windows2019/
│  ├─ README.md
│  ├─ week1/
│  ├─ week2/
│  └─ week3/
├─ windows2022/
│  ├─ README.md
│  ├─ week1/
│  ├─ week2/
│  └─ week3/
├─ compare/
├─ scripts/
├─ member/
└─ templates/
```

## Directory Guide

- `docs/`: rules, schedule, shared concepts
- `windows2019/`: weekly notes and practice for Windows Server 2019
- `windows2022/`: weekly notes and practice for Windows Server 2022
- `compare/`: week-by-week comparison notes
- `member/`: personal notes and practice logs for each team member
- `templates/`: reusable note and practice templates

## Recommended Workflow

1. Fill `docs/schedule.md` with weekly topics.
2. Write theory in `notes.md` and hands-on output in `practice.md`.
3. Store screenshots in each week's `assets/` folder.
4. Summarize differences in `compare/week*-comparison.md`.
5. Use `member/` for personal ownership and individual progress tracking.

## Suggested Commit Messages

- `docs: update study rules`
- `win2019: add week1 notes`
- `win2022: add week2 practice log`
- `compare: add week3 comparison`
- `member: add personal practice notes`
